# line-following-robot-digital-logic_No-Microcontroller
This project demonstrates a line following robot control system implemented using only digital logic ICs (no microcontroller).

#🔹 Software Implementation

This project does not use a microcontroller or firmware.

The control logic is entirely implemented in hardware using digital ICs.
All behavior is defined by logic gate combinations and latch memory elements.

Simulation and validation were performed using Proteus Design Suite.
